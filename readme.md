## ThinkAdmin for html

此分支是为`ThinkAdmin`下个版本前端`UI`构建测试用途。

目前拟定以`Vue3`为技术核心库，基础`UI`组件库使用`Element-Plus`构建。

##### **集成组件：**

* vue v3.2.26
* vue.router v4.0.12
* vue.sfc.loader v0.8.4
* vue element-plus v1.2.0-beta.6
